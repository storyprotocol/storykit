# CollectionGraph

Show IP Asset derivatives graph.

#### Example

<Canvas of={CollectionGraphStory.Select} />

<Controls />

## Dependencies

The IP Graph also requires [react-force-graph-2d](https://www.npmjs.com/package/react-force-graph-2d).

```bash
pnpm add react-force-graph-2d
```

## Usage

```ts
import { IpProvider } from "@storyprotocol/storykit"
import { CollectionGraph } from "@storyprotocol/storykit"

const ExamplePage = () => {
  return (
    <IpProvider ipId={"0xbbf08a30b9ff0f717a024a75963d3196aaf0f0dd"}>
      <CollectionGraph />
    </IpProvider>
  );
};

export default ExamplePage;
```

#### Arguments

- **width**: _number_ - The width of the canvas
- **height**: _number_ - The height of the canvas
- **darkMode**: _boolean_ - Light or Dark mode
